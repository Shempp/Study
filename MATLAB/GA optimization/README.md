# Optimization for Easom's function with Genetic Algorithm
## Function definition
`fEaso(x1,x2)=-cos(x1)·cos(x2)·exp(-((x1-pi)^2+(x2-pi)^2))`
## Global minimum
`f(x1,x2)=-1; (x1,x2)=(pi,pi)`
